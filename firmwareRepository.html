Filename format must be aa_bbccdddeee.hex
Where
aa is Controller Type (ap = 300 board; fw = 918 board)
bb is Radio Type (at = LV100/LX200; cb = LA100)
cc is System Configuration Code (number)
ddd is Revision (number)
eee is the Radio Handler (number)
